## 🎨 Die wichtigsten Skills im Detail

### ascii-art

**Auslöser:** „Mach n ASCII-Banner", „Cowsay", „pyfiglet"

Erzeugt ASCII-Art aus Text oder Bildern. Nutzt pyfiglet für Banner, cowsay für sprechende Tiere, boxes für Rahmen und image-to-ascii für Bilder.

```bash
# Banner
figlet -f big "Hermes"

# Cowsay
cowsay "Hallo Welt"

# Bild zu ASCII
jp2a foto.jpg
```

### excalidraw

**Auslöser:** „Zeichne ein Architektur-Diagramm", „Flow-Chart", „Sequenzdiagramm"

Erzeugt handgezeichnet wirkende SVG-Diagramme als Excalidraw-JSON. Perfekt für Architektur-Skizzen, Ablaufdiagramme und UML-artige Darstellungen.

**Einsatzbereiche:**
- System-Architektur
- Datenflüsse
- Sequenzdiagramme
- Wireframes

### claude-design

**Auslöser:** „Design eine Landingpage", „Mach einen Prototypen", „HTML-Artfakt"

Claude Design erstellt einmalige HTML-Artifakte – Landingpages, Präsentationsdecks, interaktive Prototypen. Alles in einer einzigen HTML-Datei mit CSS + JS.

### humanizer

**Auslöser:** „Mach den Text menschlicher", „Entferne KI-Formulierungen"

Durchkämmt Texte auf typische KI-Phrasen („zweifellos", „bemerkenswert", „es ist wichtig zu betonen") und ersetzt sie durch natürliche Formulierungen.

## 💡 Tipps

- **Excalidraw** ist perfekt für schnelle Architektur-Skizzen – sieht aus wie von Hand gezeichnet
- **popular-web-designs** ist eine Inspirations-Goldgrube: 54 echte Designsysteme als Quellcode
- **manim-video** braucht Geduld – Rendering kann Minuten dauern
- **humanizer** am Ende jedes Textes laufen lassen verbessert die Lesbarkeit massiv
